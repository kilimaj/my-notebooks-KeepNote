ifconfig - Shows different network interface types and their ip address with mac address.
iwconfig - This shows shows wireless network interface details.


ping ipaddress (icmp)
Example:
ping 192.168.15.1
This command sends connection requests to a server to know if it is reachable through the network. Its an endless non ending command hence can be stopped by Ctrl + C

arp -a : Show the ip address its talking to and the mac address associated with it. Hence id an ipadress machine thats trying to connect.

netstat -ano : Shows active connections running on your machine. Showing open ports used by active connections.

route - This prints the routing table, showing gateway where traffic exits.
