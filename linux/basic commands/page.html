pwd - print working directory, this prints the current directory

cd - change directory to go backward cd ..
    - cd directory
    - cd /directory
~ sends to home directory

ls - list contents of a directory
ls -la : this shows hidden files and folders

mkdir - make directory
mkdir john

rmdir - remove directory
rmdir john

echo "Hi!" > test.txt : this creates a file test.txt and writes Hi! inside it
echo "Am John" >> test.txt : the two >> appends the statement in the file if it was one > then it would have overwritten the statement in the file.

cp test.txt Downloads/ : this copies the file to Downloads directory, makes a copy.

rm Downloads/test.txt : removes file copied

mv test.txt Downloads/ : moves the file test.txt from current directory to Downloas directory

locate name-of-file : this cmd brings in all files that with that name in it

updatedb : Incase you dont find the file using locate then can update its database.

passwd user-name : prompts to change password

man cmd-name : manual pages brings explanation how to use a cmd also options.

cmd-name --help : This also shows available options for the cmd
